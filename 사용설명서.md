# 원소기호 타일 만들기 — 쉬운 설명서

이 폴더의 도구를 이용하면 **주기율표에서 원소를 클릭**해
3D 프린팅용 STL 파일을 한 번에 만들 수 있어요.

---

## 1. 처음 한 번만 준비하기

다음 세 가지를 컴퓨터에 설치합니다.

### ① Python (파이썬)
- 사이트: <https://www.python.org/downloads/>
- "Download Python 3.x.x" 큰 노란 버튼을 누르면 됩니다.
- **설치 화면에서 맨 아래 "Add Python to PATH" 체크박스를 꼭 켜고** 설치하세요.

### ② OpenSCAD (오픈스캐드)
- 사이트: <https://openscad.org/downloads.html>
- "Windows Installer" 항목을 받아 일반적인 방법으로 설치합니다.
- 설치 후에는 따로 실행할 필요 없습니다. 도구가 알아서 부릅니다.

### ③ 한글 글꼴 (최소 1개)
한글이 또렷이 새겨지려면 아래 폰트들 중 **최소 1개**가 깔려 있어야 합니다.
모두 **상업적 이용 가능**한 무료 폰트입니다.

- **나눔고딕 / 나눔명조 / 나눔스퀘어** — 네이버에서 "나눔폰트 다운로드" 검색
- **본고딕(Noto Sans KR)** — Google Fonts에서 검색
- **프리텐다드(Pretendard)** — github.com/orioncactus/pretendard
- **에스코어 드림** — S-Core 사이트
- **카페24 시리즈, 배민 한나/도현/주아, 고운돋움/바탕** 등

가장 무난한 선택은 **나눔고딕**입니다. 설치 후 GUI에서 자동으로 목록에 뜹니다.

## 2. 프로그램 실행 — 가장 쉬운 방법 (GUI)

### ① 폴더 열기 ⚠️ 중요
탐색기에서 **반드시 `maru-python-version` 폴더 안까지 들어가서** 엽니다.
- `element-symbol-generator` 폴더가 아니라 그 **안의** `maru-python-version` 폴더입니다.
- 폴더 안에 `타일생성기.bat` 파일이 보여야 정확한 위치입니다.

### ② 실행
**`타일생성기.bat`** 파일을 더블클릭하세요.
잠시 후 창이 뜹니다. 더 이상 검은 창(터미널)을 만질 필요가 없습니다.

> Windows에서 "이 앱이 PC에 변경 가능…" 같은 경고가 뜨면
> "추가 정보" → "실행" 으로 넘기세요. (배포처가 분명한 스크립트입니다.)

### ③ 사용법 — 4개 탭 순서대로

**① 원소 선택 탭**
- 화면 가운데에 주기율표가 보입니다.
- 원하는 원소를 **클릭**하면 파란 테두리가 생기며 선택됩니다. 다시 클릭하면 해제.
- 아래쪽에 "X개 선택됨" 으로 표시됩니다.
- "중학교 권장" 버튼을 누르면 자주 쓰는 15개를 한 번에 선택해줍니다.
- 직접 만든 CSV 파일을 쓰고 싶으면 위쪽의 **"CSV 파일 사용"** 으로 전환.

**② 글자 디자인 탭**
- **폰트** — 컴퓨터에 설치된 상용 가능 한글 폰트가 자동으로 목록에 뜹니다.
- **굵기** — Regular / Bold / ExtraBold / Light. (없는 굵기는 폰트가 알아서 대체)
- **추가로 굵게 (mm)** — 글자 외곽선을 더 굵게. 0.05~0.15 정도면 인쇄 시 또렷.
- **각 글자별 크기 (mm)** — 기호 / 이름 / 원자량 / 원자번호 각각 설정.
- **각인 깊이 (mm)** — 글자가 얼마나 도드라지게 새겨질지.

**③ 인쇄 설정 탭**
- **타일 한 변 (mm)** — 정사각형 한 변의 길이. 60~90mm 권장.
- **타일 두께 (mm)** — 너무 얇으면 휨, 두꺼우면 시간↑. 4mm 기본.
- **끼움 여유 (mm)** — 옆 타일과 끼울 때의 헐거움. 0.15 기본,
  뽑아보고 빡빡하면 +0.02, 헐겁면 -0.02씩 조정.

**④ 출력 설정 탭**
- **OpenSCAD 위치** — 보통 비어있으면 "자동 탐지" 한 번 누르면 끝.
- **결과 폴더** — STL 파일이 모이는 폴더. 기본은 `out\`.

### ④ 생성
화면 아래 큰 **🧪 STL 생성 시작** 버튼을 클릭.
- 진행 막대가 차오르며 한 개씩 만들어집니다.
- 완료되면 알림음 + 결과 폴더가 자동으로 열립니다.

### ⑤ 설정 저장 (선택)
잘 맞춘 설정은 **"프리셋 저장"** 으로 보관해두세요.
다음에 **"프리셋 불러오기"** 로 한 번에 복원할 수 있습니다.

## 3. 자주 생기는 문제

### 더블클릭했는데 잠깐 깜빡이고 사라져요
→ Python이 설치되지 않았거나 PATH 등록이 안 된 것입니다.
   1번의 ① 항목대로 Python을 설치하면서 **"Add Python to PATH"** 를 꼭 켜세요.

### 폰트 목록에 한글 폰트가 적게 보여요
→ 컴퓨터에 설치된 상용 가능 한글 폰트만 목록에 뜨도록 만들었습니다.
   1번의 ③ 항목에서 마음에 드는 폰트를 추가로 설치하면 다음 실행 때 자동으로 보입니다.

### 한글이 두부(☒☒☒) 모양으로 나와요
→ 선택한 폰트가 한글을 지원하지 않거나, 그 폰트가 사실은 설치가 안 된 상태입니다.
   **나눔고딕**으로 바꿔 보세요. 그래도 깨지면 나눔고딕 재설치.

### "OpenSCAD를 찾을 수 없다"고 떠요
→ ④ 출력 설정 탭에서 **"자동 탐지"** 버튼을 눌러 보세요.
   그래도 못 찾으면 **"찾아보기"** 로 직접 `openscad.exe` 위치를 지정.
   (보통 `C:\Program Files\OpenSCAD\openscad.exe`)

### 결과가 너무 작거나 너무 커요
→ ③ 인쇄 설정 탭에서 **타일 한 변 (mm)** 값을 바꾸세요.
   60~90mm 사이가 학교 수업용으로 적당합니다.

### 끼움이 너무 빡빡/헐거워요
→ ③ 탭의 **끼움 여유 (mm)** 값을
   빡빡하면 +0.02, 헐겁면 -0.02 씩 조정해서 다시 1개만 뽑아 시험.

## 4. 명령줄(터미널)에서 쓰는 방법 — 고급 사용자용

GUI 없이 한 번에 자동 생성하려면 PowerShell에서:

```powershell
python generate_tiles.py samples\elements_sample.csv `
  --out out `
  --openscad "C:\Program Files\OpenSCAD\openscad.exe"
```

CSV 파일은 다음과 같은 형식입니다:

```
원자번호,기호,이름,원자량
1,H,수소,1.008
2,He,헬륨,4.0026
6,C,탄소,12.011
```

엑셀로 만들 때는 반드시 **"CSV UTF-8 (쉼표로 분리) (\*.csv)"** 형식으로 저장하세요.
일반 CSV로 저장하면 한글이 깨질 수 있습니다.

## 5. 한 개만 빠르게 만들고 싶을 때

원소 한 개만 필요하면 GUI 없이도 가능합니다.
이 폴더에 `한개.scad` 라는 새 파일을 메모장으로 만들고 아래 내용을 붙여넣으세요.
**저장할 때 인코딩을 반드시 "UTF-8"로** 선택해야 한글이 안 깨집니다.

```
include <element_tiles.scad>
symbol = "Te";
name = "텔루륨";
number = "127.60";
atomic_number = "52";
```

그리고 PowerShell에서:

```powershell
& "C:\Program Files\OpenSCAD\openscad.exe" -o Te.stl 한개.scad
```

`Te.stl` 파일이 같은 폴더에 만들어집니다.

## 6. 인쇄(3D 프린팅) 팁

- 슬라이서에서 **레이어 높이 0.16~0.2 mm** 권장 (글자가 또렷)
- **타일 본체 색**과 **글자 색**을 다르게 하면 가독성이 올라갑니다
  (필라멘트 교체 또는 멀티 컬러 프린터 사용).
- 끼움쇠(connector)가 별도로 필요하면 한 번 더 OpenSCAD에서 다음 명령:
  ```powershell
  & "C:\Program Files\OpenSCAD\openscad.exe" -o connector.stl `
    -D "show_tile=false" -D "show_connector=true" element_tiles.scad
  ```
  학생 수만큼 넉넉히 뽑으세요.
